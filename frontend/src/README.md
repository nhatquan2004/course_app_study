# Frontend src

Thu muc `src` duoc tao theo structure tham khao tu `course-app2/frontend/src`.

Goi y chia module:

- `common`: components, hooks, layouts, services, utils dung chung.
- `config`: cau hinh frontend nhu API URL, route constants.
- `context`: React context/provider dung toan app.
- `modules/admin`: dashboard, courses, categories, lessons, orders, users.
- `modules/auth`: login, register.
- `modules/client`: home, courses, course-detail, checkout, my-learning.
- `styles`: style dung chung neu can tach khoi `app/globals.css`.

Git khong track thu muc rong, nen cac folder con se hien tren GitHub sau khi co file code that ben trong.
